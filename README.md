⚡ TGSPDCL Service Performance & Consumption Analysis Dashboard
📌 Project Overview

The TGSPDCL Service Performance & Consumption Analysis Dashboard is an interactive Power BI project developed to analyze electricity service performance, billing efficiency, and electricity consumption across different divisions and areas of Telangana State Southern Power Distribution Company Limited (TGSPDCL).

The project converts raw electricity service data into meaningful insights through data cleaning, transformation, and visualization. It enables users to monitor operational performance, identify billing gaps, and make data-driven decisions using interactive dashboards.

🎯 Objectives
Analyze service performance across divisions and areas.
Monitor billing efficiency using key performance indicators (KPIs).
Study electricity consumption trends across different regions.
Identify billing gaps and low-performing zones.
Create an interactive dashboard for better business insights.

📂 Dataset
Source: Telangana Open Data Portal
Records: 71,000+ electricity service records
Format: Excel / CSV

🛠️ Technologies Used
Microsoft Power BI
Power Query
Microsoft Excel
Data Cleaning
Data Transformation
Data Visualization
Business Intelligence

📊 Dashboard Features
KPI Cards for key business metrics
Division-wise and Area-wise Performance Analysis
Billing Efficiency Analysis
Electricity Consumption Analysis
Interactive Slicers and Filters
Drill-down Reports
Cross-filtering between visuals
Dynamic Charts and Tables

📈 Key Insights
Identified billing gaps across multiple divisions.
Compared service performance across different operational areas.
Analyzed electricity consumption patterns and regional demand.
Highlighted low-performing regions requiring operational improvement.
Generated actionable insights to support data-driven decision-making.

📌 Project Outcome
This dashboard provides a comprehensive view of electricity service performance and consumption trends, enabling users to quickly identify operational issues, monitor billing efficiency, and support informed decision-making through interactive visualizations.
